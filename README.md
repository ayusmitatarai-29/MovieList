# MovieList
A responsive and interactive movie listing web application built using HTML, CSS, and JavaScript. This project allows users to browse, search, and explore movies with an attractive UI inspired by modern streaming platforms.

# Features 

Real Time Movie Search Functionality 

Movie Cards With Posters, titles, and Details 

Filter Movies by Year and Category 

Fully Responsive Design 

Clean and Modern UI

# Technologies Used 

HTML5 

CSS3

JavaScript
